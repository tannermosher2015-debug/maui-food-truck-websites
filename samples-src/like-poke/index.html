One truck, one island

We don't ship the fish in. We catch it here.

Like Poke? is Daniel Kalahiki's truck on Lono Ave in Kahului - built on one simple idea: the fish should taste like Maui, not like a freezer. That means locally caught, cut fresh, and sauced by hand.

Word got around. The bowls landed on Food Network, the line got longer, and the reviews piled up - but it's still the same truck, the same fish, the same question every regular answers the same way.

Like poke? Yeah. You will.
